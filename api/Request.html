Name Type Attributes Default Description

an ordered list of one or more SIP URIs to proxy the request to
remainInDialog boolean <optional>
 false

if true add a Record-Route header and emain in the SIP dialog after the INVITE transaction.
followRedirects boolean <optional>
 false

if true respond to 3XX redirect responses by generating a new INVITE to the SIP URI in the Contact header of the response
forking string <optional>
 sequential

'simultaneous' or 'sequential'; dicates whether the proxy waits for a failure response from one target before trying the next, or forks the request to all targets simultaneously
provisionalTimeout string <optional>

amount of time to wait for a 100 Trying response from a target before trying the next target; valid syntax is '2s' or '1500ms' for example
finalTimeout string <optional>

amount of time to wait for a final response from a target before trying the next target; syntax is as described above for provisionalTimeout
